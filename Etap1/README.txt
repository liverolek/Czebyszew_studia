Program do obliczenia przybliżonej wartości całki n-tego wielomianu Czebyszewa ze wzoru trapezów.
Program wylicza wartość początkowo metodą sekwencyjną, a następnie metodą równoległą wraz z wizualizacją czasów obliczeń.

Instrukcja:
Wartości dla następujących parametrów nalezy wprowadzić do pliku dane.txt w folderze ConsoleApp1:
- n - stopień wielomian Czebyszewa
- a - początek przedziału całki 
- b - koniecz przedziału całki
- N - ilość podprzedziałów całki
